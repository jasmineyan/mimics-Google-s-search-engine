Instructions to run:

Frontend:
1. In the folder, run python Alexgle.py
2. This will bring up the home page(url: http://localhost:8080/)
3. Type in the keywords and hit submit
5. Then will bring you to the result page showing the word count
6. Results for the top 20 keywords will be placed in an HTML table with name 'history' in the text file history.txt


Backend; 
1. In the backend folder , run python crawler.py to see the output of the content crawled from the three website in urls.txt
2. In the same folder , run the crawler_tester  , to initiate an auto test . Which will return the passed/ failed test cases.
